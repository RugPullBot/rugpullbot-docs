## Key Features

### Token Creation
- **AI-Powered Metadata**: "Let AI Handle It" button auto-fills token info from detected rugs
- **Real Historical Data**: Create coins based on actual proven market patterns
- **Social Links**: Add Twitter, Telegram, and website links to your tokens
- **Custom Images**: Upload token images or use images from original rugs

### Live Trading
- **Real Price Action**: TradingView charts with live candlestick data
- **Bundler Configuration**: Set dev buy amounts and bundler wallet configurations
- **Multiple Rug Triggers**: Market cap, holder count, or liquidity-based triggers
- **Real-time Statistics**: Track PnL, market cap, bonding curve progress, and more

### Wallet Management
- **Multiple Wallets**: Create and manage multiple Solana wallets
- **Balance Tracking**: Real-time balance updates with USD conversion
- **Transfer System**: Send SOL between wallets with ease
- **Withdrawal Configuration**: Set up multiple withdrawal addresses with percentage splits

### Security Features
- **Anti-MEV Protection**: Protect transactions from sandwich attacks
- **Anti-Sniper Mode**: Prevent known sniper bots from front-running
- **2FA Authentication**: Optional two-factor authentication for account security
